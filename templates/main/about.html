- About Me -

My name is Pius Lucky, I'm a software developer and researcher based in Nigeria (Benin-City). I build apps in Python(Django), Nodejs, React and other technologies.


I created this blog to share some of my insights and experience about web technologies and programming in general while adhering to some stiff principles of Clean code conventions, Unit-testing, Modularization etc. I simply want to transform you into a professional in the field of web development.


The blog is called joint-access-programmer denoting a person that has an expert idea (access) about majority (joint) of relevant technologies (programs). We aim to achieve that here.
